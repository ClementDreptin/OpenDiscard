# OpenDiscard - desktop app
OpenDiscard Desktop App is an executable version of [OpenDiscard](https://github.com/ClementDreptin/OpenDiscard/) for Windows, MacOS and Linux.

### Technologies used
- [JavaScript](https://developer.mozilla.org/en-US/docs/Web/JavaScript)
- [Node.js](https://nodejs.org/)
- [Electron](https://www.electronjs.org/)
- [Vue.js](https://vuejs.org/)
